# Визуализатор графа зависимостей пакетов

## Назначение проекта
Инструмент для анализа и графического представления зависимостей между программными пакетами. Поддерживает работу с официальными репозиториями и локальными тестовыми наборами данных.

## Технические характеристики

### Поддерживаемые режимы работы
- Реальный режим (анализ зависимостей из PyPI)
- Тестовый режим (работа с локальными JSON-файлами)

### Интерфейс командной строки

#### Обязательные параметры:
- `--package` - указывает имя анализируемого пакета

#### Дополнительные параметры:
- `--repo` - адрес репозитория или путь к файлу (по умолчанию: https://pypi.org)
- `--mode` - выбор режима работы: real или test
- `--output` - имя файла для сохранения результатов
- `--ascii` - флаг включения текстового представления
- `--filter` - строка для исключения пакетов

## Инструкция по использованию

### Базовые команды запуска

Стандартный анализ пакета:
```bash
python 1etap.py --package requests
```
